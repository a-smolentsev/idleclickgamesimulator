# Idle Game Clicker

Простая инкрементная игра с элементами симулятора


# Возможности

 - Магазин покупок для прокачки персонажа
 - Прибавление денег во время отсутствия
 - Есть параметры у игрока как голод и отдых
 - Игрок будет голодать во время вашего отсутствия
 - Прокачка персонажа (смена одежды, дома)

## Стек

 - Room в качестве БД для хранения всех значений у игрока
 - Корутины
 - Hilt для инъекции зависимостей
 - MVVM модель
